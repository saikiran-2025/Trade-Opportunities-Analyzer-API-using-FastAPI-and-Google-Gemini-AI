# Trade Opportunities Analyzer API using FastAPI and Google Gemini AI
Trade Opportunities API is a FastAPI-based backend service that analyzes Indian market sectors and provides AI-generated trade opportunity insights. The system collects recent market news related to a given sector, processes the information using Google Gemini AI, and generates a structured Markdown report highlighting potential trade opportunities, market trends, and associated risks. Users can provide a sector name such as banking, pharmaceuticals, technology, or agriculture, and the API returns an AI-generated market analysis report based on the latest available news data.

This project is developed using Python and FastAPI as the backend framework for building REST APIs. Market data is collected through web scraping using Requests and BeautifulSoup, while Google Gemini API is used for AI-based analysis of the collected news information. The application implements JWT authentication using python-jose, environment variable management with python-dotenv, and rate limiting using SlowAPI to enhance security and prevent API abuse. The project dependencies include FastAPI, Uvicorn, Requests, BeautifulSoup4, Google-GenerativeAI, Python-JOSE, Passlib (bcrypt), SlowAPI, Python-Multipart, and Python-Dotenv, which together enable API development, security, AI integration, and data processing.
